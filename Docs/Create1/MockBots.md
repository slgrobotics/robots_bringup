## MockBots initiative

There's a project initiated by members of the HomeBrew Robotics club to create instructions for building a modern version of Turtlebot.
The "MockBots" can be built using different vacuum cleaner *bases* - Create, Roombas, Neato etc.
They can use different sensors.

The builder's experiences are described in the book available on [Amazon](https://www.amazon.com/MockBOT-Over-shoulder-instructions-personal/dp/B0GK1TS39N).

## MockBots and *articubot_one* codebase

While the book's instructions offer introduction to ROS2 and walk you through initial robot setup, the reader has to do a lot of extra work to bring his code to a manageable state.
This is where using a codebase with already organized collection of *config*, *launch* and *description* files starts making sense.

The articubot_one repository already has *Turtlebot* - a robot very similar to MockBots.
